信号说明
- 发生某种事件的通知机制
- 异步行为，中断进程执行的位置是不可预测的
- 信号是不排队
- sigaction函数具备灵活性和控制性

可重入(reentrant)函数
- 同一个进程的多线程可以安全调用执行的函数

async-signal-safe函数说明
- 从信号处理器函数调用时，保证其实现是安全的
- 一个函数可重入，或者无法中断，那么它是异步信号安全的
