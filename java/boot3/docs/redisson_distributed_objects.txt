监听对象事件类概述
- 读操作后创建/更新的数据: org.redisson.api.listener.TrackingListener 
- 数据已经创建或是更新: org.redisson.api.listener.SetObjectListener
- 对象超时: org.redisson.api.ExpiredObjectListener
- 对象删除: org.redisson.api.DeletedObjectListener

监听代码示例
RBucket<String> set = redisson.getBucket("anyObject");
int listenerId = set.addListener(new SetObjectListener() {
     @Override
     public void onSet(String name) {
        // ...
     }
});
int listenerId = set.addListener(new DeletedObjectListener() {
     @Override
     public void onDeleted(String name) {
        // ...
     }
});
// ...
set.removeListener(listenerId);

RateLimiter示例
// 同步操作
RRateLimiter limiter = redisson.getRateLimiter("myLimiter");
// 仅仅初始化一次
// 2秒5次
limiter.trySetRate(RateType.OVERALL, 5, 2, RateIntervalUnit.SECONDS);
// 获取三个许可
limiter.acquire(3);
// 异步操作
RRateLimiter limiter = redisson.getRateLimiter("myLimiter");
RFuture<Boolean> setRateFuture = limiter.trySetRate(RateType.OVERALL,
    5, 2, RateIntervalUnit.SECONDS);
RFuture<Void> aquireFuture = limiter.acquire(3);
